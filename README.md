# 🍕 Food Delivery Time Prediction

## 🚀 Project Overview

This Streamlit web application predicts food delivery times using machine learning. It analyzes factors like distance, weather, traffic, and preparation time to provide accurate estimates, enhancing operational efficiency and customer satisfaction.

## ✨ Key Features & Methodology

*   **Exploratory Data Analysis (EDA):** Visual and statistical analysis (Pearson, ANOVA) of `Delivery_Time_min` against numerical and categorical features.
*   **Data Preprocessing:** Missing value imputation and Label Encoding for categorical data.
*   **Machine Learning Model:** Utilizes a **Random Forest Regressor** for robust predictions.
*   **Hyperparameter Tuning:** Optimized model performance using `RandomizedSearchCV` with cross-validation.

## 🛠️ Technologies Used

Python, Streamlit, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, SciPy.

## 🚀 Getting Started

1.  Clone repository:
    `git clone https://github.com/Ansh5922/DeliverIQ.git`
2.  Install dependencies:
    `pip install -r requirements.txt`
3.  Run app:
    `streamlit run app.py`
